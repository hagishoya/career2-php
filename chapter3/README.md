## Gitによるバージョン管理

Linux感覚でWindowsが操作できる。

[Git for Windows](https://gitforwindows.org/)

対象プロジェクトでこのコマンド

```
git config user.name "ユーザー名"
git config user.email "メールアドレス"
```

本当はこのコマンド

```
git config --global user.name "ユーザー名"
git config --global user.email "メールアドレス"
```
